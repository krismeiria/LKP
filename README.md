This project collects and structures data on *Lembaga Kursus dan Pelatihan (LKP)* across Indonesia from the official portal:  
https://kursus.kemendikdasmen.go.id

The workflow consists of:

- **Web scraping**
  - Collects institutional profiles and attributes
- **OCR-based document processing**
  - Extracts program info from *SK Operasional* files
- **LLM-powered classification**
  - Clusters extracted text into 10 predefined program categories
- **Data transformation & consolidation**
  - Produces analytics-ready dataset
- **Tableau dashboards**
  - Visualizes data for regional and program-level analysis
